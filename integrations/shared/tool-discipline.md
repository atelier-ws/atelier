## Tool discipline

Lead with Atelier's code-intel tools — `explore` (call graph: definitions, callers, callees, and usages in one call) and `search` (semantic code search). No host tool does this; prefer them over `git log` / `rg` archaeology when getting oriented in unfamiliar code.

Host tools are disabled and fail with "No such tool available" — use the Atelier tool instead: `Bash` → `bash`, `Read` → `read`, `Grep` / `Glob` → `grep`, `Edit` / `Write` → `edit`.
