# Milestone v0.5 Archive: Quality & Benchmark Lift

**Status:** Shipped  
**Archived:** 2026-06-01  
**Phase Range:** 22-27

## Milestone Summary

`v0.5` shipped the quality-and-maintainability foundation for Atelier. The milestone closed the lint/coverage gate work, removed silent broad-except debt, cleaned stdout/logging behavior, and completed the CLI decomposition. The original benchmark-expansion/result-publication work in Phases 26-27 was not carried forward as originally planned; it was superseded by the real-proof program moved into `v0.6`.

## Key Accomplishments

1. Enabled enforceable quality gates for BLE001/T20 plus slow-inclusive coverage workflows.
2. Burned down silent broad-except debt across touched gateway/runtime surfaces.
3. Removed non-CLI stdout leaks and normalized intentional CLI output behavior.
4. Finished the CLI decomposition into thin gateway registration plus command modules.
5. Re-scoped benchmark publication work into the later world-class proof milestone instead of shipping smoke-only claims.

## Detailed Phase Notes

### Phase 22 - Lint + Coverage Gates

Shipped stricter lint gating for broad exceptions and print debt, plus reproducible full-suite coverage commands and nightly coverage automation.

### Phase 23 - Silent Exception Audit

Removed or narrowed silent `except Exception` handling and tightened the associated gateway/tool-handler coverage around the touched surfaces.

### Phase 24 - Stdout to Logging

Converted non-CLI stdout usage to logging/stderr-safe behavior and preserved intentional CLI output through the approved user-facing paths.

### Phase 25 - CLI Decomposition

Completed the split of the CLI into a thin Click registration layer plus command modules, while keeping command names and user-facing help behavior stable.

### Phase 26 - A/B Suite Expansion

Did not ship in the milestone's final form. The benchmark-expansion plan was superseded by the later, honest-proof program that now lives in `v0.6`.

### Phase 27 - Public Results Publication

Did not ship in the milestone's final form. Public-results publication was intentionally deferred until the real-proof surfaces in `v0.6` could replace smoke-only or circular benchmark claims.
