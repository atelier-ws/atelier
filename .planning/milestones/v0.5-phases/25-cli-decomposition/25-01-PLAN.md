<objective>
Establish the Phase 25 safety net and the command-package substrate **before any command
group moves**. This plan adds (1) a recursive `--help` command-tree renderer/invariant test
that later slices use for live before/after comparisons, (2) an MCP-tool-only suppression regression test, and
(3) the `cli/commands/` package scaffolding (`__init__.py` register aggregator, `_dev.py`
dev-gate scaffolding, `_shared.py` CLI-only glue) wired back into `app.py`.

Purpose: Lock the byte-identical CLI contract (QBL-CLI-04) so every later extraction slice can
compare a live pre-edit help tree to its live post-edit help tree without committing a fixture
generated from unrelated dirty WIP, and break the future `app.py`↔`commands/*` circular-import risk by giving command
modules a downward import target (`_dev.py`, `_shared.py`).
Output: Two new tests, the `cli/commands/` package skeleton, and an `app.py` that imports the
relocated glue/dev scaffolding while preserving the recursive help tree.
</objective>
