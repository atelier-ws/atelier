  <action>
    Create `src/atelier/infra/code_intel/languages.py` as a stdlib-only leaf module
    with `from __future__ import annotations`. It MUST import nothing from
    `atelier.core` (import-cycle Pitfall 1).

    Define a frozen dataclass `Language` with fields `name: str`,
    `extensions: frozenset[str]`, `parser_name: str`, `scip_indexer: str | None`.
    `name` equals the tree-sitter-language-pack parser name; `parser_name` usually
    equals `name`.

    Define `LANGUAGES: tuple[Language, ...]` seeded as the UNION of the
    `capability.py::_language_for` map and `search_read.py::_LANG_MAP` extension
    coverage. Dropping any extension is a regression (Pitfall 3). Required entries
    (extensions lowercase, leading dot):
    - python (.py, .pyi) → scip_indexer "scip-python"
    - typescript (.ts, .tsx) → scip_indexer "scip-typescript"
    - javascript (.js, .jsx, .mjs, .cjs) → scip_indexer "scip-typescript"
    - bash (.sh, .bash, .zsh) → scip_indexer None  [canonicalizes shell→bash at the data layer per DLS-LANG-03]
    - csharp (.cs) → None  [canonical spelling is `csharp`, NOT `c_sharp`]
    - go (.go), rust (.rs), java (.java), kotlin (.kt, .kts), scala (.scala),
      ruby (.rb), cpp (.cpp, .cc, .cxx, .hpp, .hh), c (.c, .h), swift (.swift),
      php (.php), sql (.sql), markdown (.md, .markdown), yaml (.yaml, .yml),
      toml (.toml), json (.json) → all scip_indexer None
    Seed `scip_indexer=None` for everything except the three known indexers
    above; the full SCIP table belongs to Phase 19 (do not pre-populate it here).

    Derive at import time: `EXTENSION_TO_LANGUAGE: dict[str, Language]` (comprehension
    over `LANGUAGES`), `_BY_NAME: dict[str, Language]`, and
    `ALL_LANGUAGES: frozenset[str] = frozenset(_BY_NAME)`.

    Define `language_for_path(path: str | Path) -> Language | None` returning
    `EXTENSION_TO_LANGUAGE.get(Path(path).suffix.lower())`, and
    `language_by_name(name: str) -> Language | None` returning `_BY_NAME.get(name)`.

    Export via `__all__`: `Language`, `LANGUAGES`, `EXTENSION_TO_LANGUAGE`,
    `ALL_LANGUAGES`, `language_for_path`, `language_by_name`. Do NOT add any
    auto-detection machinery — the registry is a static table.
  </action>
