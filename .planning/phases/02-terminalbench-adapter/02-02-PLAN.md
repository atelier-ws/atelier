Purpose: The adapter wires the Atelier bench-mode toggle into TerminalBench's Docker execution model. `run_terminalbench_trial()` is the central function: it instantiates Harness, runs the trial, then post-processes TrialResults + stream.jsonl to assemble the full transcript. **parse_stream_jsonl() function:**
Define `def parse_stream_jsonl(log_path: Path) -> dict[str, Any]:` that:
1. Reads `log_path.read_text().splitlines()`
2. Skips empty lines; tries `json.loads(line)` for each, ignores JSONDecodeError
3. Finds the last line where `obj.get("type") == "result"` — assigns to `result_line`
4. If no result line found, returns `{"error": "no_result_line"}` with all numeric fields as 0
5. Returns dict with keys: `input_tokens`, `output_tokens`, `cache_creation_input_tokens`, `cache_read_input_tokens`, `total_cost_usd`, `duration_ms`, `duration_api_ms`, `num_turns`, `is_error`, `stop_reason`
6. Each field uses `.get(key, default)` with defaults: 0 for ints/floats, False for bool, "" for str

**AtelierClaudeAgent class:**
Define `class AtelierClaudeAgent(AbstractInstalledAgent):` with:

- `__init__(self, bench_mode: str = "on", model: str | None = None, **kwargs):`
  - Calls `super().__init__(**kwargs)`
  - Sets `self._bench_mode = bench_mode` and `self._model = model`

- `@staticmethod def name() -> str:` returns `"atelier-claude"`

- `@property def _env(self) -> dict[str, str]:`
  - Builds a MINIMAL env dict (NOT full os.environ — prevents dev contamination)
  - Must include: `ANTHROPIC_API_KEY` (from os.environ, raise KeyError if missing), `ATELIER_BENCH_MODE` (self._bench_mode), `FORCE_AUTO_BACKGROUND_TASKS` ("1"), `ENABLE_BACKGROUND_TASKS` ("1")
  - If self._model is set: add `ANTHROPIC_MODEL = self._model`
  - Elif `ANTHROPIC_MODEL` in os.environ: forward it
  - Must NOT include: ATELIER_DEV_MODE (never forward — causes tool visibility contamination per PITFALLS.md #3b), CURSOR_WORKSPACE_ROOT, VSCODE_CWD
  - Return the dict

- `@property def _install_agent_script_path(self) -> Path:`
  - Returns `self._get_templated_script_path("setup.sh.j2")`
  - This uses AbstractInstalledAgent's built-in method to find `benchmarks/terminalbench/setup.sh.j2`
  - **IMPORTANT**: `setup.sh.j2` MUST exist in the same directory as `agent_adapter.py` or `FileNotFoundError` is raised
  - Template content (Jinja2): install nvm → Node 22 → `@anthropic-ai/claude-code@{{ version | default('latest') }}`

5. Inside try/finally, set up run:

   - `trial_out = out_dir / run_id`
   - Instantiate Harness with `agent_import_path="terminalbench.agent_adapter:AtelierClaudeAgent"`, `agent_kwargs={"bench_mode": bench_mode, "model": model}`, `output_path=trial_out`, `run_id=run_id`, `dataset_name=dataset_name`, `dataset_version=dataset_version`, `task_ids=[task_id]`, `n_attempts=1`, `n_concurrent_trials=1`, `cleanup=True`

   - Parse stream log: call `parse_stream_jsonl(stream_files[0])` if stream_files else `{"error": "no_stream_log"}`
