<objective>
Deliver the default-definition layer that Eval has implicitly through embedded defaults, adapted to Atelier's generated multi-host surfaces.

Purpose: satisfy EXEC-12 and DFLT-01 through DFLT-04 before the benchmark solver runtime depends on these definitions.

Output: canonical defaults registry, non-overwriting bootstrap helpers, renderer/check integration, and tests proving generated surfaces are synced.
</objective>
